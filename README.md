
# VCEW

A Flask studio and equipment ordering management website.
this project was written to meet the real demands of the college's real studio 

## Features

- Login & Register system

- User ordering system 

- AWS database

- Easy equipment tracking

- Studio and podcast room orders by time

- Notifications system

- Forgot my password by email 
 
- Out of order marking and back in

- Jenkins pipeline support

- Runnable testing

## Users
The system has 3 types of users in the veterinary clinic:
* Manager

* Student

* Teacher

## Deployment

- Install the dependencies in the requirements.txt file
- run the flask upp with
```bash
  flask run
```

## Note
Check with system owner that the database is up and running
## Running Tests

To run tests:

```bash
  coverage run -m pytest
```
